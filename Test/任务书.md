## 项目任务书

### 一、项目名称

**基于多 Worker 协同的乳腺癌诊断机器学习实验平台设计与实现**

### 二、项目背景

在人工智能辅助分析场景中，传统机器学习方法具备部署轻量、训练高效、结果可解释性较强等优势。相比深度学习，传统机器学习对硬件和环境依赖较低，更适合在课程项目、原型验证与快速演示场景中使用。scikit-learn 官方提供了多个经典分类数据集，其中乳腺癌 Wisconsin 诊断数据集是一个经典且易于上手的二分类数据集，包含 569 个样本和 30 个数值特征，适合用于分类建模、特征分析与可视化展示。

本项目拟构建一个面向实验教学与智能体协同演示的轻量级实验平台，以“任务委托—脚本生成—结果绘图—报告生成”为核心流程，展示多 Worker 协作完成机器学习实验任务的全过程。

### 三、项目目标

1. 构建一个基于传统机器学习的二分类实验流程。
2. 实现从数据加载、预处理、模型训练、性能评估到结果可视化的完整闭环。
3. 设计多 Worker 协同机制，使不同 Worker 分工完成代码编写、图表生成和实验报告整理任务。
4. 自动生成实验结果图与 PDF 实验报告，提高实验流程自动化程度。
5. 为后续扩展到更多课程实验、更多数据集和更多智能体协同任务提供基础框架。

### 四、项目内容

本项目主要包括以下内容：

#### 1. 数据集选取与处理

选取 scikit-learn 自带的 Breast Cancer Wisconsin Diagnostic 数据集作为实验对象。该数据集包含 569 个样本，其中恶性 212 个、良性 357 个，共 30 个正实数特征，适用于二分类建模。

主要任务包括：

- 数据加载与结构检查
- 训练集/测试集划分
- 特征标准化处理
- 标签分布统计

#### 2. 机器学习建模

实现并比较多种传统机器学习分类模型，例如：

- 逻辑回归
- 支持向量机
- 随机森林
- K 近邻

通过统一流程对模型进行训练、验证与对比，选出综合表现较优的模型。

#### 3. 结果评估与可视化

生成并整理以下实验结果：

- 分类准确率
- Precision / Recall / F1-score
- 混淆矩阵
- ROC 曲线与 AUC
- 特征重要性图
- 模型对比柱状图

#### 4. 多 Worker 协同实验流程

设计 3 类核心 Worker：

- **Script Worker**：负责根据任务目标自动生成和完善训练脚本
- **Plot Worker**：负责根据实验输出自动绘制图表并保存素材
- **Report Worker**：负责汇总实验结果、图表和结论，自动生成 PDF 实验报告

#### 5. 自动化实验报告生成

基于实验结果与可视化图表，自动生成一份结构完整的实验报告 PDF，内容包括：

- 项目背景
- 数据集说明
- 实验方法
- 实验结果
- 结果分析
- 结论与展望

### 五、技术路线

本项目采用“轻量环境 + 自动化流水线”的实现思路，具体技术路线如下：

1. 使用 Python 作为主要开发语言
2. 使用 pandas 进行数据处理
3. 使用 scikit-learn 完成模型训练与评估
4. 使用 matplotlib 进行图表绘制
5. 使用 reportlab 或 matplotlib + PDF 导出方案生成实验报告
6. 使用多 Worker 协同调度方式组织任务执行流程
