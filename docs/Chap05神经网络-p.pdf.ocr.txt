第五章 神经网络。

page 2
主要内容：5.1 神经网络简介，5.2 误差逆传播算法 BP，5.3 其他常见神经网络模型，5.4 深度学习关联。

page 3
神经网络是由大量简单神经元互连组成的并行分布式网络。它能够通过学习从样本中获得知识，并用连接权值保存知识。

page 4
M-P 神经元模型由输入、连接权重、阈值和激活函数构成。神经元接收多个输入信号，计算加权和并与阈值比较，再通过激活函数产生输出。

page 5
激活函数用于引入非线性。常见激活函数包括阶跃函数、Sigmoid 函数等。Sigmoid 函数把输入压缩到 0 到 1 之间，常用于表示神经元兴奋程度。

page 6
多层前馈神经网络包含输入层、隐含层和输出层。相邻层之间全连接，同层神经元之间不存在连接。多层网络可以通过隐含层学习复杂的非线性映射。

page 7
神经网络发展经历过兴奋期、沉寂期和热潮。BP 算法推动多层网络训练，深度学习引发新的热潮。

page 10
误差逆传播算法 BP 用于训练多层前馈神经网络。给定训练集，网络根据当前权重计算输出，将输出误差从输出层向隐含层反向传播，利用梯度下降更新连接权和阈值。

page 11
BP 算法推导中，网络输出为 y_j，误差为 E_k。BP 是基于梯度下降的迭代学习算法，以最小化训练误差为目标。

page 13
Sigmoid 函数的导数可以写作 f'(x)=f(x)(1-f(x))。BP 推导中输出层神经元的梯度项 g_j 与输出误差和激活函数导数有关。

page 15
BP 算法输入训练集、学习率和网络结构。每轮训练先前向计算输出，再反向计算梯度项，最后更新连接权和阈值，直到达到停止条件。

page 16
标准 BP 算法每次只针对一个训练样例更新连接权和阈值，累积 BP 算法先读取整个训练集，累积误差后再更新参数。标准 BP 更新频繁，累积 BP 更新稳定但可能较慢。

page 17
缓解过拟合的主要策略包括早停 early stopping 和正则化 regularization。早停使用验证集，如果验证误差连续上升则停止训练；正则化在误差目标中增加描述网络复杂度的惩罚项。

page 18
神经网络目标函数常存在多个局部极小。训练结果对参数初始化敏感，不同初始参数可能导致不同局部极小点。常用方法包括多组初始值、模拟退火和随机扰动等。

page 21
其他常见神经网络模型包括 RBF 网络、ART 网络、SOM 网络、级联相关网络、Elman 网络和 Boltzmann 机。

page 22
RBF 径向基函数网络是一种单隐层前馈神经网络。隐层神经元使用径向基函数作为激活函数，输出层通常是线性组合。训练通常包括确定中心、宽度以及输出层权值。

page 23
SOM 自组织映射网络是一种竞争学习网络，能够将高维输入映射到低维拓扑结构中，常用于聚类和可视化。

page 25
Elman 网络是一种递归神经网络，具有承接层或上下文层，可以保存前一时刻隐层状态，因此适合处理时间序列和动态系统建模。

page 27
深度学习兴起与人工神经网络发展密切相关。卷积神经网络 CNN、深度信念网络 DBN、生成对抗网络 GAN、GPT 等都是深度学习的重要代表。

page 30
卷积神经网络 CNN 通过卷积层、池化层和全连接层提取局部特征，常用于图像识别、手写数字识别等任务。

page 31
深度学习通常需要大量数据和强计算能力。预训练加微调可以提升学习能力，逐层预训练、权共享和 dropout 等技巧有助于训练深层网络。

page 32
Dropout 是深度学习常用技巧，训练时随机丢弃部分神经元连接，可以减少过拟合。ReLU 激活函数 f(x)=max(0,x) 也是常用技巧之一。

page 33
传统机器学习通常依赖人工设计特征再学习分类器，深度学习可以直接从数据中学习特征表示，并联合优化特征学习和分类模型。
