第六章 支持向量机。

主要内容：6.1 间隔与支持向量，6.2 对偶问题，6.3 核函数，6.4 软间隔与正则化，6.5 支持向量回归，6.6 核方法。

page 3
线性模型试图在样本空间中寻找一个超平面，将不同类别的样本分开。

page 5
在多个可分超平面中，应该选择正中间、容忍性好、鲁棒性强、泛化能力最强的超平面。

page 6
划分超平面可表示为 w^T x + b = 0。样本点到超平面的距离与 |w^T x + b| / ||w|| 有关。支持向量是距离分离超平面最近的训练样本。

page 7
支持向量机的目标是寻找参数 w 和 b，使几何间隔最大。最大化 2/||w|| 等价于最小化 1/2 ||w||^2，并满足 y_i(w^T x_i + b) >= 1。

page 8
支持向量机基本型是一个带约束的凸优化问题。目标函数是 1/2 ||w||^2，约束条件是 y_i(w^T x_i + b) >= 1。凸优化问题局部最优就是全局最优。

page 14
等式约束优化问题可构造拉格朗日函数 L(x,mu)=f(x)+mu h(x)。在最优点处梯度满足一阶必要条件。

page 16
不等式约束 g(x)<=0 定义可行域，等式约束 h(x)=0 可看作高维空间中的曲面。约束优化问题需要在可行域内寻找极小点。

page 22
KKT 条件是约束优化的核心条件。优化问题是凸优化时，KKT 条件既是极小值点的必要条件也是充分条件。KKT 条件中的互补松弛要求约束不活跃时对应拉格朗日乘子为零。

page 31
如果有 m 个不等式约束，需要面对 2^m 种活跃约束组合，因此需要找到新的方法，这引出对偶方法。

page 33
支持向量机的对偶问题通过拉格朗日乘子 alpha_i 表示。对偶形式最大化关于 alpha 的目标函数，并满足 sum alpha_i y_i = 0 和 alpha_i >= 0。

page 34
解的稀疏性：训练完成后，大部分样本对应的 alpha_i 为 0，只有 alpha_i > 0 的样本对模型有贡献，这些样本就是支持向量。

page 41
SMO 是 Sequential Minimal Optimization，序列最小优化算法。基本思想是每次只选择两个变量 alpha_i 和 alpha_j 更新，其余变量固定，从而把大优化问题分解为小优化问题。

page 42
SMO 变量选择原则包括选择违反 KKT 条件程度最大的变量，以及选择能使目标函数变化最大的另一个变量。

page 44
当样本在原始空间线性不可分时，可以通过非线性映射把样本映射到高维特征空间，使其在高维空间线性可分。
支持向量机处理线性不可分问题的核心方法是核技巧：先用非线性映射 phi(x) 将输入样本映射到高维特征空间，再在高维特征空间中寻找线性分类超平面。

page 45
支持向量机在特征空间中的模型依赖内积。若可以用核函数替代高维空间内积，就无需显式计算映射后的高维特征。
因此核函数可以避免显式计算高维映射 phi(x)，直接用 k(x_i,x_j) 表示样本在特征空间中的内积。

page 46
核函数 k(x_i,x_j)=phi(x_i)^T phi(x_j)。核技巧通过直接计算核函数，避免显式构造高维特征空间。
核函数 k(x_i,x_j) 是支持向量机解决非线性分类的重要工具。

page 47
常用核函数包括线性核、多项式核、高斯核 RBF、拉普拉斯核、Sigmoid 核。Mercer 定理说明只要一个对称函数对应的核矩阵半正定，它就能作为核函数。

page 50
软间隔允许某些样本不满足约束，以提高模型泛化能力。软间隔引入松弛变量和惩罚参数 C，在间隔最大化和训练误差之间折中。

page 51
0/1 损失函数不连续、非凸，数学性质不好，因此支持向量机通常使用替代损失函数。

page 52
常见替代损失包括 hinge 损失、指数损失和对率损失。支持向量机使用 hinge 损失 max(0,1-z)。

page 53
软间隔支持向量机的原始问题是在 1/2||w||^2 基础上加 C sum xi_i，约束 y_i(w^T x_i+b)>=1-xi_i，xi_i>=0。C 是正则化常数。

page 58
正则化把经验风险和模型复杂度结合起来。L2 范数对应岭回归和支持向量机，L1 范数对应 LASSO，可用于特征选择。

page 61
支持向量回归 SVR 假设允许预测值与真实值之间最多有 epsilon 的偏差。只有落在 epsilon 间隔带之外的样本才计算损失。

page 63
SVR 的 epsilon 不敏感损失函数在误差绝对值不超过 epsilon 时损失为 0，超过 epsilon 后损失线性增长。

page 70
表示定理说明支持向量机模型可以表示为训练样本的核函数线性组合。模型只依赖支持向量及其系数。

page 71
核线性判别分析 kLDA 是将线性判别分析通过核技巧扩展到非线性问题的方法。

page 72
Take Home Message：支持向量机的最大间隔思想，对偶问题及稀疏解，通过向高维空间映射解决线性不可分问题，引入软间隔缓解不可分，支持向量回归处理回归问题，核方法推广到其他学习模型。
